# 👥 User Roles & Permissions

## Role Hierarchy

```
Admin (Full Control)
  ↓
Moderator (Manage Content & Reports)
  ↓
User (Regular User)
```

---

## User Role

**Default role for all new users.**

### Permissions

✅ Create posts (text, images)
✅ Create anonymous posts
✅ Comment on posts
✅ React to posts/comments
✅ Send private messages
✅ Join group chats
✅ Report inappropriate content
✅ Block/unblock users
✅ Update own profile
✅ Browse public feed

---

## Moderator Role

**Manually assigned by Admin to trusted users.**

### All User Permissions +

✅ View all reports
✅ Review reported content
✅ Remove inappropriate posts/comments
✅ Send warnings to users
✅ Access moderation dashboard

---

## Admin Role

**Full system control.**

### All Permissions

✅ Full access to all user data
✅ Manage user roles
✅ Ban/unban users
✅ Delete users
✅ View all reports
✅ Manage reports
✅ Delete posts/comments
✅ View audit logs
✅ Access admin dashboard
✅ Manage moderators
✅ System settings
✅ Analytics & metrics
